## Requirements
+ Windows
+ Node (tested on 12.x 10.x 8.x)
+ windows-build-tools
## Instalation
Make sure you've installed all the necessary build tools
```
npm install --global windows-build-tools
```
Then execute
```
npm install
npm start
```
